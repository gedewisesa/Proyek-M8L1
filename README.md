# Proyek-M8L1
M8L1

Bot discord klasifikasi musim.
Proyek ini berguna untuk mengklasifikasi musim menggunakan gambar dengan sarana discord.
Proyek ini mengusung tema "Perubahan Iklim"
Musim yang dapat diklasifikasi antara lain:
1. Musim panas
2. Musim dingin
3. Musim hujan
4. Musim semi
5. Musim gugur
6. Musim kemarau
